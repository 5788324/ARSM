## 一、项目概览

ARSM 是一个私人音频图书馆，目标是把本地音声资源与 asmr.one 采集能力结合起来，形成“可采集、可导入、可搜索、可播放、可续听、可评分、可管理”的长期使用系统。

| 项目 | 信息 |
|------|------|
| 仓库 | github.com/5788324/ARSM |
| 技术栈 | Next.js 16 + TypeScript + Prisma 5 + SQLite + NextAuth v5 + Tailwind 4 |
| 当前阶段 | Phase 6 已完成 |
| 当前重点 | Phase 7 打磨与扩展 |

---

## 二、已实现功能

### 2.1 采集与导入
- asmr.one Provider
- AcquisitionJob + runner
- inspect → download → import → metadata 全链路
- 文件级下载进度
- 本地文件夹导入
- 重复检测与 review

### 2.2 元数据
- 自动补全 asmr.one 元数据
- 圈子 / 声优 / 标签关联
- 管理员手动编辑标题、原标题、发售日
- 管理员手动重抓 asmr.one

### 2.3 播放与字幕
- 全局播放器
- 队列 / 循环 / 倍速 / 续播
- 曲目自动分组
- 文本字幕阅读
- `.lrc/.vtt/.srt` 时间轴解析
- 跟播高亮与点击跳转

### 2.4 浏览与用户数据
- 作品库搜索、排序、筛选、分页
- URL 状态同步
- 社团 / 声优 / 标签关系页
- 首页最近添加、最近采集、最近播放、统计
- 收藏
- 收听历史
- 个人评分（按用户独立）

### 2.5 部署与运行
- Windows 启动/停止批处理
- Docker 基础支持
- 深色模式

---

## 三、当前限制

- 采集来源仍以 asmr.one 为主
- 普通用户界面仍可进一步隐藏管理员字段
- 字幕暂不支持翻译、双语、AI 生成
- 统计能力还不够完整
- 尚无 Android / 桌面打包壳

---
