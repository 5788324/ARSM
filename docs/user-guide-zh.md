# ARSM 用户指南

## 快速开始

### 启动

1. 找到项目文件夹 `G:\Hermes Agent\ARSM\ARSM`
2. 双击 **`启动ARSM.bat`**
3. 等待浏览器打开 `http://localhost:3000`
4. 登录：用户名 `admin`，密码 `admin`

### 停止

- 双击 **`停止ARSM.bat`**，或关闭启动时的命令行窗口

---

## 如何使用

### 1. 从 asmr.one 采集作品

1. 登录后进入「管理」
2. 打开「采集任务」
3. 输入 RJ 编号，例如 `RJ01538000`
4. 保持「自动导入」勾选
5. 点击「开始采集」
6. 等待任务跑完
7. 回到「作品库」或首页查看作品

### 2. 导入本地文件

1. 管理 → 导入媒体
2. 输入本地文件夹路径，例如 `C:\Users\YANG\Music\arsm.one`
3. 点击「开始导入」

### 3. 播放与继续收听

- 打开作品详情页后可直接播放全部或单曲
- 底部全局播放器支持队列、循环、倍速
- 首页会显示最近播放，方便继续收听

### 4. 字幕使用

- 详情页可打开字幕文件
- `.lrc/.vtt/.srt` 支持跟播高亮与点击跳转
- `.txt/.pdf` 目前按文本阅读处理

### 5. 评分与元数据

- 所有登录用户都可以给作品打个人评分
- 管理员可以编辑标题、原标题、发售日，并重新抓取 asmr.one 元数据

## 常见问题

### 启动失败？

- 确认 Node.js 已安装
- 确认 3000 端口未被其他程序占用
- 查看 `logs/` 下的日志文件

### 登录不了？

- 先用 `admin / admin`
- 如有改动账号，请使用当前数据库中的账号

### 评分为什么和管理员编辑分开？

- 评分是每个用户自己的数据
- 标题、发售日这类元数据属于全局数据，只允许管理员修改

### 字幕为什么有时不能跟播？

- 只有 `.lrc/.vtt/.srt` 这类带时间轴的字幕才支持跟播
- `.txt/.pdf` 目前是纯文本阅读模式
