# ARSM 用户指南

## 快速开始

### 启动

1. 找到项目文件夹 `G:\Hermes Agent\ARSM\ARSM`
2. 双击 **`启动ARSM.bat`**
3. 等待浏览器打开 `http://localhost:3000`
4. 登录：用户名 `admin`，密码 `admin`

### 停止

- 双击 **`停止ARSM.bat`**

---

## 如何使用

### 1. 从 asmr.one 采集作品

1. 登录后进入「管理」
2. 打开「采集任务」
3. 输入 RJ 编号，例如 `RJ01538000`
4. 保持「自动导入」勾选
5. 点击「开始采集」
6. 等待任务跑完
7. 回到「作品库」或首页查看作品

### 2. 导入本地文件

1. 管理 -> 导入媒体
2. 输入本地文件夹路径，例如 `C:\Users\YANG\Music\arsm.one`
3. 点击「开始导入」

### 3. 搜索作品

- 作品库支持关键词搜索
- 支持排序：最新、标题、曲目数、时长
- 支持高级筛选：
  - 排除词
  - 最低时长 / 最高时长
  - 社团
  - 声优
  - 标签
  - 带字幕
- 筛选状态会同步到地址栏

### 4. 播放与继续收听

- 作品详情页可直接播放全部或单曲
- 底部全局播放器支持队列、循环、倍速
- 首页显示最近播放
- 「继续收听」页显示：
  - 播放次数
  - 收听作品数
  - 总收听时长
  - 最常播放
  - 最近播放记录

### 5. 字幕使用

- 详情页可打开字幕文件
- `.lrc/.vtt/.srt` 支持跟播高亮与点击跳转
- `.txt/.pdf` 目前按文本阅读处理

### 6. 评分与元数据

- 所有登录用户都可以给作品打个人评分
- 管理员可以编辑标题、原标题、发售日，并重新抓取 asmr.one 元数据
- 普通用户不会看到管理员编辑字段

## 常见问题

### 启动失败？

- 确认 Node.js 已安装
- 确认 3000 端口未被其他程序占用
- 查看 `logs/` 下的日志文件

### 登录不了？

- 先用 `admin / admin`
- 如有改动账号，请使用当前数据库中的账号

### 评分为什么和管理员编辑分开？

- 评分是每个用户自己的数据
- 标题、发售日这类元数据属于全局数据，只允许管理员修改

### DLsite 为什么不能下载？

- 当前只接通了元数据探测能力
- 还没有接入真实下载链路
