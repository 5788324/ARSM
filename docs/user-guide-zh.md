# ARSM 用户指南

## 快速开始

### 启动

1. 找到项目文件夹 `G:\Hermes Agent\ARSM\ARSM`
2. 双击 **`启动ARSM.bat`**
3. 等待黑色窗口出现 `✓ Ready`
4. 浏览器自动打开 `http://localhost:3000`
5. 登录：用户名 `admin`，密码 `admin`

### 停止

- 双击 **`停止ARSM.bat`**，或直接关闭黑色命令行窗口

> 首次启动会稍慢（需要编译），之后热更新会快很多。

---

## 如何使用

### 1. 从 asmr.one 采集作品

1. 登录后点顶栏「管理」
2. 点绿色卡片「📥 采集任务」
3. 输入 RJ 编号（如 `RJ01538000`）
4. 保持「自动导入」勾选
5. 点「开始采集」
6. 等待任务完成（inspect → download → import）
7. 回到「作品库」查看新作品

### 2. 导入本地文件

1. 管理 → 📁 导入媒体
2. 输入本地文件夹路径（如 `C:\Users\YANG\Music\arsm.one`）
3. 点「开始导入」

### 3. 播放音频

- 作品库 → 点击封面 → 进入详情页
- 每个曲目右侧有播放器
- **注意**：当前每个曲目独立播放，暂无全局播放器

## 常见问题

### 启动失败？

- 确认 Node.js 已安装
- 确认端口 3000 未被占用（启动脚本会自动清理）
- 查看日志文件 `logs/arsm-server.err.log`

### 登录不了？

- 确认使用 `admin` / `admin`
- 尝试重新运行 `启动ARSM.bat`

### 下载的作品导入后拆成很多个？

- 这是旧版 bug，已通过 `groupByTop` 修复
- 如需重建库：运行 `scripts/rebuild-library.ps1`

### 数据库在哪？

- `prisma/arsm.db`（SQLite 文件）
- 备份在 `prisma/backups/`
