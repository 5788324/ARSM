# asmr.one 功能对标分析

日期：2026-06-20

对比 asmr.one 网站与 ARSM 当前实现，列出可优化/新增的功能。

---

## 一、播放器

### asmr.one 有，ARSM 没有

| 功能 | asmr.one | ARSM 现状 | 优先级 |
|------|----------|-----------|--------|
| 全局播放器 | 页面顶部固定播放器，切换页面不中断 | 每首曲目独立的 `<audio>` 标签 | **P1** |
| 连续播放 | 播完自动跳到下一首 | 单独播放，无自动续播 | **P1** |
| 播放队列 | "添加到播放列表"按钮 | 无 | P2 |
| 进度记忆 | "标记进度" — 记录收听位置 | 有 `ListeningHistory` 但页面未展示续听入口 | P2 |
| 播放速度控制 | 播放器有变速选项 | 无 | P3 |
| 文件树导航 | 可折叠的 ROOT → 文件夹 → 文件 树 | 扁平曲目列表 | P2 |

### 建议实现

- **P1 全局播放器**：React Context + 底部固定播放栏，展示当前曲目、进度条、上一首/下一首
- **P1 连续播放**：曲目列表末尾自动播放下一个
- **P2 播放队列**：右键曲目 → 加入队列（类似 Spotify）

---

## 二、字幕功能

### asmr.one 有，ARSM 没有

| 功能 | asmr.one | ARSM 现状 |
|------|----------|-----------|
| "带字幕"筛选 | 作品列表页有独立 checkbox | 无 |
| 多语种标签 | 作品卡片显示"带字幕"/"多语种"标记 | 无 |
| 字幕内容展示 | 播放器内同步显示字幕文本 | 无 |

### asmr.one 的字幕来源

- asmr.one 本身不提供 AI 字幕，标签基于作品元数据中的 `has_subtitle`
- 字幕内容：部分作品带日文 VTT/SRT/TXT

### 建议实现

| 功能 | 阶段 | 说明 |
|------|------|------|
| `hasSubtitle` 字段展示 | P1 | `InspectResult` 已有该字段，前端显示"带字幕"标签 |
| 作品列表"带字幕"筛选 | P2 | 新增 API 参数过滤 |
| 字幕文件下载与展示 | P3 | 下载后解析 VTT/SRT 显示在播放器下方 |
| AI 字幕生成 | P4 | Whisper 转写 → LLM 翻译，已在 postprocess 留接口 |

## 三、浏览与发现

### asmr.one 有，ARSM 没有

| 功能 | asmr.one | ARSM 现状 | 优先级 |
|------|----------|-----------|--------|
| 评分系统 | 1-5 星评分，显示评分人数 | 无 | P2 |
| 语言筛选 | 下拉选择"中文"等语言 | 无 | P2 |
| 热门作品 | 首页"🔥 热门作品"横滑 | 无 | P3 |
| 分页 | 底部有分页控件 | `take: 50` 无翻页 | P2 |
| VA 可点击 | 点击声优名筛选该 VA 作品 | 声优只显示文本 | P2 |
| 社团可点击 | 点击社团名筛选该社团作品 | 社团只显示文本 | P2 |
| 排序 | 下拉选择排序方式 | 固定 `updatedAt desc` | P3 |

### 建议实现

- **P2 评分**：Work 表加 `rating` 字段，前端星星组件
- **P2 VA/社团可点击**：Link 到 `/?va=xxx` 或 `/works?circle=xxx`
- **P2 分页**：`GET /api/works?page=1&pageSize=20`

---

## 四、作品详情页

### asmr.one 有，ARSM 没有

| 功能 | asmr.one | ARSM 现状 |
|------|----------|-----------|
| DLsite 价格/销量 | 显示"1430 JPY 销量: 6922" | 无 |
| 评分与评论 | 星星评分 + 评论数 | 无 |
| 下载按钮 | 每个文件旁有下载按钮 | 无直接下载，只能流式播放 |
| 文件树 | 可折叠 ROOT/子文件夹 | 扁平列表 |
| 添加到播放列表 | 按钮 | 无 |
| 标记进度 | 按钮 | 无（有 ListeningHistory 但无 UI） |

### 建议实现

- **P2 下载**：`<a href="/api/tracks/{id}/download" download>` — 直接下载
- **P2 文件树**：前端读取 tracks API 的 folder 结构，渲染可折叠树
- **P3 价格/销量**：inspect 时从 DLsite/asmr.one 获取并存储

---

## 五、功能优先级总览

### P1 — 显著提升体验，建议下一轮做

1. **全局持续播放器** — 底部固定栏，切换页面不中断
2. **自动连续播放** — 曲目播完自动跳到下一首
3. **"带字幕"标签展示** — 作品卡片和详情页显示

### P2 — 完善产品力

4. VA/社团 可点击筛选
5. 作品列表分页
6. 评分系统（1-5 星）
7. 播放队列 / 添加到播放列表
8. 文件树导航（替代扁平曲目列表）
9. 直接下载按钮
10. 语言筛选

### P3 — 锦上添花

11. 播放速度控制
12. 热门作品推荐
13. DLsite 价格/销量信息
14. 评论系统

### P4 — 后续大功能

15. AI 字幕生成（Whisper + LLM 翻译）
16. 字幕同步展示
