## 当前版本现状

**已交付**: MVP+，核心采集链路完整（asmr.one → 本地 → 导入 → 播放）

| 能力 | 状态 |
|------|------|
| 从 asmr.one 采集 | ✅ 流式下载 + 自动导入 |

## 路线图

### Phase 1 (✅ 已完成) — MVP 骨架
- Next.js 项目脚手架
- 认证系统
- 基本页面（首页/作品库/收藏/管理）
- 导入系统（本地扫描）

### Phase 2 (✅ 已完成) — 后台补全
- 元数据 fetch/apply
- 重复审查
- 任务监控
- 仓库管理
- 全站中文化

### Phase 3 (✅ 已完成) — 采集系统
- asmr.one Provider
- Provider registry + runner
- AcquisitionJob 后端任务化
- 共享 import service
- 采集任务后台

### Phase 4 (📋 下一阶段) — 播放体验
- 采集进度可视化
- 曲目子文件夹分组
- 全局播放器（底部固定栏）
- 元数据自动匹配
- 字幕/台本识别

### Phase 5 (📋 规划中) — 产品化
- VA/社团点击筛选
- 评分系统
- 分页搜索
- 深色模式切换
- 用户文档完善

### Phase 6 (⏸️ 远期) — 智能化
- AI 字幕生成（Whisper + LLM）
- 多 provider 接入
- 播放列表/队列管理
- 移动端优化
