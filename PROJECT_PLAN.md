# ARSM 项目计划

日期：2026-06-21 | 当前阶段：Phase 9 已验收通过，Phase 10 待执行

---

## 路线图

### Phase 1（已完成）- MVP 骨架

Next.js 基础架构、认证、作品页、管理页、本地导入初版。

### Phase 2（已完成）- 后台补全

元数据、重复审查、任务监控、仓库管理、中文化。

### Phase 3（已完成）- 采集系统

asmr.one provider、registry、runner、AcquisitionJob、共享 import service。

### Phase 4（已完成）- 播放与字幕基础

采集进度、曲目分组、全局播放器、自动元数据匹配、字幕识别。

### Phase 5（已完成）- 产品化增强

搜索、筛选、排序、分页、URL 同步、关系页、队列、深色模式。

### Phase 6（已完成）- 体验深化

时间轴字幕、详情页重构、首页聚合、管理员编辑、个人评分。

### Phase 7（已完成）- 角色与测试

角色分离、测试补强、统计口径修复、DLsite stub 预留。

### Phase 8（已完成）- 来源与统计深化

DLsite inspect-only、PlayLog 累计时长链路、播放统计链路收口。

### Phase 9（已完成并验收）- 搜索深化与统计页验收闭环

DLsite inspect 结果落库、provider 标签、高级搜索、最常播放、启动脚本同步数据库结构，并完成真实页面验收闭环。

### Phase 10（下一阶段）- 产品收口

统计页产品化、采集页产品化、搜索体验深化、运行信息可视化、验收体系化。

### Phase 11（远期）- 生态化

字幕翻译 / AI 字幕、第二真实下载来源、PWA / Android / 桌面壳评估。
